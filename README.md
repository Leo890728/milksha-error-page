# 迷克夏錯誤頁面設計

## 設計風格
創造一個視覺上吸引人且具互動性的錯誤頁面，讓使用者在遇到錯誤時仍能感受到網站的品牌風格。例如文字的草地背景、品牌的顏色代表以及牛奶等等。另外可以透過與畫面上的乳牛互動寄送信件。
